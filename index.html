Abstract & Problem Statement

Problem Statement

Traditional home security systems are expensive, complex to install, and lack integration with existing smart TVs. Users need multiple remotes for different devices, and there is no unified solution for TV protection combined with home automation.

Proposed Solution

VisionGuard provides a cost-effective, unified IoT solution that combines Smart TV protection (AI-based threat detection), universal IR remote control, and Arduino-based sensor automation in a single mobile application.

Objectives

  • Develop real-time Smart TV threat detection system
  • Create universal IR remote for 10+ device types
  • Build Arduino sensor network for home security
  • Implement OTP-based secure authentication

📥 IR Code Capture - Arduino Code

#include <IRremote.h>
int recvPin = 15;
IRrecv irrecv(recvPin);
decode_results results;
void setup() { Serial.begin(9600); irrecv.enableIRIn(); }
void loop() { if (irrecv.decode(&results)) { Serial.println(results.value, HEX); irrecv.resume(); } }

Sample Captured Codes: Power: 0xFFA25D, Vol+: 0xFF629D, Vol-: 0xFFE21D

📊 Sensor Data Reading

#include <DHT.h>
#define DHTPIN 4
DHT dht(DHTPIN, DHT11);
int pirPin = 14;
void setup() { Serial.begin(9600); dht.begin(); pinMode(pirPin, INPUT); }
void loop() {
  float temp = dht.readTemperature();
  int motion = digitalRead(pirPin);
  Serial.print("Temp: "); Serial.print(temp); Serial.print("°C | Motion: "); Serial.println(motion);
  delay(2000);
}

⚡ Device Control via Relay

int relayPin = 5;
void setup() { pinMode(relayPin, OUTPUT); Serial.begin(9600); }
void loop() {
  if(Serial.available()) {
    char cmd = Serial.read();
    if(cmd == '1') digitalWrite(relayPin, HIGH);
    if(cmd == '0') digitalWrite(relayPin, LOW);
  }
}
